Github

গিটহাব শিখতেছি, কিছু কিছু কমান্ড মুখস্থ করা লাগবে দেখা যাচ্ছে,

  • git add .
  • git commit -m"---"
  • git push

    • গিটে ডাটা পুশ করার জন্য উপরের গুলো

  • git branch
    • ব্র্যাঞ্চ চেক করার জন্য
  • git branch (name)
    • নতুন ব্র্যাঞ্চ ক্রিয়েট করার জন্য
  • git checkout -b (branch name)
    • নতুন ব্র্যাঞ্চ ক্রিয়েট করে সেই ব্রাঞ্চে একটিভ হবার জন্য নতুন ব্রাঞ্চ ক্রিয়েট করলেই শুধু হবে না সেটাকে গিটহাবে আপলোড (পুশ) করার জন্য একটা কমান্ড দিতে হবে।
