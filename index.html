Github

গিটহাব শিখতেছি, কিছু কিছু কমান্ড মুখস্থ করা লাগবে দেখা যাচ্ছে,

  • git add .
  • git commit -m"---"
  • git push

    • গিটে ডাটা পুশ করার জন্য উপরের গুলো

  • git branch
    • ব্র্যাঞ্চ চেক করার জন্য
  • git branch (name)
    • নতুন ব্র্যাঞ্চ ক্রিয়েট করার জন্য
  • git checkout -b (branch name)
    • নতুন ব্র্যাঞ্চ ক্রিয়েট করে সেই ব্রাঞ্চে একটিভ হবার জন্য
  • নতুন ব্রাঞ্চ ক্রিয়েট করলেই শুধু হবে না সেটাকে গিটহাবে আপলোড (পুশ) করার জন্য একটা কমান্ড দিতে হবে। :git push --set-upstream origin (branch name)
  • নতুন ক্রিয়েট করা ব্রাঞ্চকে Main Branch এ এড করার জন্য প্রথমে main branch এ আসতে হবে, কমান্ড হচ্ছে git checkout main এরপর git merge (যেই ব্রাঞ্চকে এড করতে হবে সেটার নাম)

    • গিটহাবে যা যা ভুল হতে পারে

    1. দেখা যাবে vscode এ সেভ না করেই গিটহাবে পুশ করলাম কিন্তু পুশ হবে না কারণ আমিতো সেভই করি নাই
    2. আরেকটা ভুল হতে পারে, যদি আমি অথবা অন্য কেউ কোন নতুন ব্র্যাঞ্চ ক্রিয়েট করে এখন তার সেই ব্রাঞ্চ দেখার জন্য বা এডিট করার জন্য git checkout (branch name) দিলেই কিন্তু হবে না কারণ আমার vscode তো গিটহাব সার্ভারের এই চেঞ্জ জানে না আমাকে এই জন্য যা করতে হবে তা হচ্ছে আগে git pull করে নিতে হবে তাহলে নতুন যা যা আছে সব আমার vscode এ ইম্পোর্ট হয়ে যাবে
